# Implementation

## Machine Learning Implementation

1. Data Loading:
- Loan dataset loaded using Pandas

2. Data Preprocessing:
- Missing values handled
- Categorical data encoded

3. Model Training:
- Logistic Regression
- Decision Tree
- Random Forest
- Gradient Boosting
- XGBoost
- AdaBoost

4. Model Selection:
- AdaBoost selected based on highest accuracy (~78%)

5. Model Saving:
- Saved using Joblib as .pkl files

## Web Application

- Developed using Streamlit
- Integrated ML model
- Real-time prediction
