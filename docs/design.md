# System Design

## System Architecture

The system follows a simple architecture:

User Input → Data Processing → ML Model → Prediction → Output Display

## Components

1. Frontend:
- Built using Streamlit
- Collects user input

2. Backend:
- Python-based ML model
- Handles prediction logic

3. Model:
- AdaBoost algorithm

## Input Parameters
- Gender
- Age
- Income
- Loan Amount
- Loan Term
- Credit History
- Employment Type
- Property Area

## Output
- Loan Approval Status
- Risk Percentage
- EMI Details
- Financial Suggestions
